Key Features HIGHLIGHTS
Merkle‑Verified Licenses

Tamper‑proof license generation using Merkle trees with SHA‑256 hashing and proof verification.

ECDSA Signatures

Digital signatures for license integrity and authenticity verification.

SQLite Storage

Zero external dependencies — offline‑ready local storage with SQLite.
